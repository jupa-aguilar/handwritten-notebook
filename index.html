    Google Cloud Vision API key

    Handwriting transcription uses Google Cloud Vision. Your key is stored only in this browser (localStorage) and is sent directly to Google — never to any other server. In Google Cloud Console: enable the Cloud Vision API, then create an API key under APIs & Services → Credentials. The first 1,000 pages/month are free.

    Sync across devices (optional)

    Keeps notebooks in sync through a hidden app folder in your own Google Drive. In the same Cloud Console: create an OAuth client ID of type Web application and add this site's address under Authorized JavaScript origins. Paste the client ID here, save, and sign in when prompted. Then use the same ID on every device. The Mac app also needs the client secret (plus http://127.0.0.1:17987 under Authorized redirect URIs) so it can stay signed in; browsers can leave it empty.
